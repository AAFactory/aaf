  1. 이지 패스워드

    GitHub

    이지 패스워드는 여러개의 계정과 패스워드를 하나의 잠금패턴을 이용하여 관리하는 패스워드 관리용 어플리케이션입니다.

    웹사이트, 신용카드, 도어락등의 패스워드를 관리합니다.

Awesome Manager

  1. 어썸 매니저

     

    어썸 매니저는 유소년 스포츠 클럽 관리용 프로그램으로 사용자 유형에 따라 특화된 기능을 제공 합니다.
